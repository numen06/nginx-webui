# Nginx WebUI 1.1.1

**发布日期**：（发布时填写）

## 本版亮点与改进

- 将默认 Nginx 源码版本升级至 1.31.2。
- 同步更新 Docker 镜像构建、本地预编译脚本、后端初始化流程和前端安装向导中的默认 Nginx 版本。

## 问题修复

- 修正各运行路径中默认 Nginx 版本不一致的风险。

## 升级说明

### Docker / Docker Compose

重新构建镜像并启动容器：

```bash
docker compose build --pull
docker compose up -d
```

### 源码 / 本地运行

拉取本版本代码后重新构建前端；默认源码包已替换为 `backend/default-nginx/nginx-1.31.2.tar.gz`。

## 注意事项与破坏性变更

**无**。现有数据与配置格式不受影响。
